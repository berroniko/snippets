snippets.access_large_csv.access_large_csv(filepath: str, fieldname: str) → str[source]

Yields elements from csv-file.

Opens csv-file and provides the content as a generator yielding the elements of the column fieldname. It is able to handle files larger than the available memory.

Parameters

  • filepath – path to the file to be read

  • fieldname – key of the column to be returned

Yields

Elements of the selected columns. One row per call
